제이쿼리 문서객체모델(DOM) - 내용추가, 삭제

객체 편집 메소드는 html문서안에 사용자가 선택한 요소의 앞쪽이나 뒤쪽에 내용을 추가하거나 복제,삭제하는 메소드

  • append(); - 선택한요소(부모)의 안쪽으로 맨뒤에 내용을 추가
  • prepend(); - 선택한요소(부모)의 안쪽으로 맨앞에 내용을 추가
  • after(); - 선택한 요소(형제)의 뒤에 내용을 추가
  • before(); - 선택한 요소(형제)의 앞에 내용을 추가
  • insertBefore(); - insertAfter(); - 자리배치하기

내용추가 메소드는 이메일추가,쇼핑몰 콘텐츠 '더보기',보달윈도, 갤러리 작품더보기, 공지사항 더보기,fAQ더보기 등 내용을 추가하고자 할때 응용가능

A insertAfter(b), A insertBefor(B)를 사용하면 A라는 요소의 앞쪽에 B라는 요소를 배치할 수 있고B라는 요소의 뒤쪽에 A요소를 배치할 수도 있다. after(), before()메소드와 기능은 같지만 사용법이 다르다.

예)
    $('A').after('B'); -A요소의 바깥쪽으로 뒤에 B내용을 추가

    $('B').insertAfter('A'); - A요소의 바깥쪽으로 뒤에 B내용을 추가

내용삭제하기(Remove)

사용자가 선택한 요소를 찾아서 내용만 지우거나, 태그요소 자체를 삭제할 수 있다.

  • empty() : 사용자가 선택한 요소의 내용만 삭제
  • remove() : 사용자가 선택한 요소를 삭제
