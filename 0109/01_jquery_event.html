제이쿼리 01.이벤트(event)

한번에 두개이상의 이벤트 사용가능. 웹문서에서 버튼을 클릭한다거나 마우스를 움직이거나 하는 모든 행위를 '이벤트'라고 한다. 그리고 이벤트 발생시 함수의 실행문이 수행되도록 이벤트를 지정하는 것을 '이벤트 핸들러'

이벤트의 종류

  • 마우스 이벤트 - 클릭,더블클릭,오버,아웃
  • 키보드 이벤트 - keypress
  • 포커스 이벤트 - focus...
  • 폼양식 이벤트 - submit...
  • 윈도우 이벤트 - scroll..

이벤트 등록방법(작성법)

  1. 단독이벤트 - $('이벤트 대상').이벤트메서드(function{실행할 내용;}
  2. 그룹이벤트 - $('이벤트 대상').on('이벤트 종류1, 이벤트 종류2...',function(){실행할내용}

마우스, 폼양식, 윈도우 이벤트 사용하기

  1. click - 마우스로 대상을 클릭하였을 때 이벤트 내용이 실행됨.
  2. dblclick - 마우스로 대상을 더블클릭하였을 때 이벤트 내용이 실행.
  3. mouseover(mouseenter) - 마우스로 대상을 오버하였을 때 이벤트 내용이 실행.
  4. mouseout(mouseleave) - 마우스로 대상을 오버햐였다가 뺐을 때 이벤트 내용이 실행.
  5. hover(mouseover + mouseout) - 마우스로 대상을 오버하였을때와 아웃하였을때 둘다 각각 따로 이벤트 내용을 실행
  6. focus - 마우스로 클릭하여 포커스가 발생되었을 때 이벤트 발생.(폼양식)
  7. blur - 폼양식에서 포커스가 해제되었을 대 이벤트가 발생.
  8. toggle - 대상요소를 클릭할 때 마다 지정된 함수를 호출하여 실행.
  9. each - 개별 요소 마다 함수를 각각 처리함.
  10. scroll - 윈도우 이벤트로서 가로스크롤과 세로스크롤이 동작하면 이벤트가 발생됨.(사용하면 반응형 레이아웃 만들기 가능)
